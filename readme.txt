== Description ==

<Strong>Product Dropdown For Contact Form 7</Strong> Making Product List Field to you contact form. This field creates a dropdown of current user's product.

Be sure that every product has selected category and tag to get product.

<h3>FEATURES:</h3>

<ul>
<li>Required field</li>
<li>Display product using product id(eg: 23-29)</li>
<li>Select product category and tag</li>
<li>Allow multiple product selection</li>
<li>setting id and class of the product dropdown field (optional)</li>
</ul>
